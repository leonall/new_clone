# 智谱AI（HKEX: 2513）估值分析报告

**报告日期**：2026年1月  
**股票代码**：HKEX: 2513  
**研究类型**：首次覆盖（Initiating Coverage）

---

## 一、估值方法概览

本报告采用多种估值方法对智谱AI进行综合分析：

1. **DCF估值（主要方法）**：基于财务模型预测的自由现金流进行折现估值
2. **可比公司分析（P/S倍数）**：参考国内外AI/大模型公司估值水平
3. **先例交易分析**：参考近期AI领域融资和并购交易估值
4. **估值足球图**：综合各方法得出估值区间

## 五、关键催化剂

### 5.1 正面催化剂

1. **GLM-5模型发布**（2026年Q1已发布）
   - 745B参数规模，达到国际前沿水平
   - 多模态能力大幅提升
   - 企业客户付费意愿增强

2. **海外市场拓展**
   - 东南亚、中东市场布局
   - 海外收入占比提升目标
   - 国际化战略推进

3. **商业化加速**
   - 企业客户数量突破20万
   - API调用量指数级增长
   - 行业解决方案深化

4. **盈利拐点**
   - 2027-2028年有望实现盈利
   - 毛利率持续优化
   - 运营效率提升

### 5.2 负面催化剂

1. **竞争加剧**
   - 科技巨头价格战
   - 模型能力差距缩小
   - 市场份额流失风险

2. **监管风险**
   - AI监管政策趋严
   - 数据安全合规要求
   - 出口管制影响

3. **盈利压力**
   - 持续高研发投入
   - 算力成本上升
   - 亏损周期延长

## 七、估值结论

### 7.1 核心观点

1. **DCF估值合理**：基于Base Case的DCF估值为9.5美元/股，反映公司中长期价值
2. **P/S倍数参考**：参考可比公司，10-15x P/S为合理区间
3. **成长溢价**：作为中国首家上市大模型公司，享有一定的稀缺性溢价
4. **风险提示**：高增长伴随高风险，需密切关注竞争格局和技术迭代

### 7.2 投资建议

**评级：买入（BUY）**
**目标价：8.6美元（62人民币）**
**投资期限：12-18个月**

**推荐理由：**
- 技术领先：GLM架构差异化优势明显
- 商业化领先：企业客户基础扎实
- 成长空间大：大模型市场快速增长
- 稀缺标的：中国首家上市大模型公司

**风险提示：**
- 持续亏损压力
- 竞争加剧风险
- 监管政策不确定性
- 地缘政治风险
