# 文档解析结果交叉对比分析提示词

## 任务概述
你是一个专业的文档解析结果分析专家。你需要分析三个不同文档解析工具（Qwen3、Kimi、PaddleOCR VL 1.5）对同一张图片（sample10.JPG，一份财务报表图片）的解析结果，并进行详细的交叉对比分析，找出差异点。

## 输入文件说明
你需要分析以下三个文件：
1. **qwen3-sample10.md** - 使用 Qwen3-VL-32B-Instruct 模型解析的结果（Markdown格式）
2. **kimi-sample10.md** - 使用 Kimi k2.5 模型解析的结果（Markdown格式）
3. **paddleocr-sample10.json** - 使用 PaddleOCR VL 1.5 模型解析的结果（JSON格式，包含布局信息）

所有文件都是对同一张财务报告图片（2024年年度报告第83页）的解析结果。

## 分析要求

### 1. 文本内容对比
- 对比三个文件中提取的文字内容是否一致
- 特别注意数字、表格数据、特殊符号的准确性
- 查找拼写错误、遗漏文本、额外文本
- 分析断句和段落划分的差异

### 2. 表格数据对比
- 对比所有表格的结构和内容
- 特别注意以下表格：
  - (a) 业务分部的收入信息
  - (b) 按区域披露的收入信息  
  - 8 收入
  - 客户合同收入确认时点分析表格
- 对比表格中的数值数据，注意千位分隔符的正确性
- 检查合计行的计算是否正确
- 注意脚注和说明文字的完整性

### 3. 布局和结构对比
- 对比三个工具对页面布局的理解
- 分析栏位划分、标题级别、段落结构
- 比较版面信息的描述准确性
- 检查页码、页眉、页脚信息的提取

### 4. 格式和标记对比
- 对比 Markdown 格式的处理差异
- 检查表格标记、标题格式、列表格式
- 分析特殊字符和编码处理

### 5. 错误和差异分类
对于发现的差异，按以下类别分类：
- **严重错误**：数据错误、关键信息缺失、计算错误
- **次要错误**：格式错误、轻微文本差异、布局误解
- **工具特性差异**：不同工具的输出风格差异
- **精度差异**：数值精度、文本提取精度差异

## 分析注意事项
1. **仔细核对**：每个差异点都需要在三份文件中确认
2. **关注数据准确性**：特别是财务数据，小数点、千位分隔符的准确性很重要
3. **考虑上下文**：某些差异可能是合理的（如不同的Markdown格式化方式）
4. **客观评价**：基于事实进行分析，避免主观偏好
5. **提供证据**：对于重要差异，提供具体的文本片段作为证据
